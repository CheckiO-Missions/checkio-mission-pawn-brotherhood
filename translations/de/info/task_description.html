Fast jeder auf der Welt kennt das alte Spiel Chess und hat zumindest ein Grundverständnis für seine Regeln. Es verfügt über verschiedene Einheiten mit einer breiten Palette von Bewegungsmustern, die für eine große Anzahl von möglichen unterschiedlichen Spielpositionen (zum Beispiel Number of possible chess games at the end of the n-th plies.) Für diese Mission werden wir die Bewegungen und das Verhalten von Schachfiguren untersuchen.

Schach ist ein Strategiespiel für zwei Spieler, das auf einem karierten Spielbrett gespielt wird, das in acht Reihen (Ränge genannt und mit den Zahlen 1 bis 8 bezeichnet) und acht Spalten (als Felder bezeichnet und mit den Buchstaben a bis h bezeichnet) unterteilt ist. Jedes Feld des Schachbretts wird durch ein eindeutiges Koordinatenpaar identifiziert - einen Buchstaben und eine Zahl (z.B. "a1", "h8", "d6"). Für diese Mission müssen wir uns nur mit den Bauern beschäftigen. Ein Bauer kann eine gegnerische Figur auf einem Feld diagonal vor ihm auf einer angrenzenden Datei schlagen, indem er auf dieses Feld zieht. Für weiße Bauern sind die vorderen Felder Quadrate mit einer höheren Zeilennummer als das Feld, das sie gerade besetzen.

Ein Bauer ist im Allgemeinen eine schwache Einheit, aber wir haben 8 von ihnen, die wir nutzen können, um eine Bauernverteidigungsmauer zu bauen. Bei dieser Strategie verteidigt ein Bauer die anderen. Ein Bauer ist sicher, wenn ein anderer Bauer eine Einheit auf diesem Feld schlagen kann. Wir haben mehrere weiße Bauern auf dem Schachbrett und nur weiße Bauern. Sie sollten Ihren Code so gestalten, dass er herausfindet, wie viele Bauern sicher sind.

Sie erhalten eine Reihe von Feldkoordinaten, auf denen wir weiße Bauern platziert haben. Sie sollen zählen, wie viele Bauern sicher sind.

Eingabe: Die Koordinaten der platzierten Bauern in Form von Strings.

Ausgabe: Die Anzahl der sicheren Bauern als Ganzzahl.
