Quase todo mundo conhece o antigo jogo de xadrez e tem pelo menos conhece as regras básicas. O jogo tem diferentes unidades, com uma grande variedade de movimento que permitem que haja um grande número de posições possíveis nos jogos diferentes (por exemplo Número de possíveis jogos de xadrez ao término da n-ésima jogada. ) Para esta missão, vamos examinar os movimentos e comportamento dos peões de xadrez.

O xadrez é um jogo de estratégia em que dois jogadores utilizam um tabuleiro dispostos em oito linhas (identificadas com números de 1 a 8) e oito colunas (identificadas com letras A a H) de quadrados. Cada quadrado do tabuleiro é identificado pelo seu próprio e único par de coordenadas — uma letra e um número (ex: "A1", "h8", "d6"). Para esta missão, só vamos trabalhar com os peões. Um peão pode capturar uma peça adversária em um quadrado na diagonal da frente, em uma coluna adjacente, se movendo para este quadrado. Os peões brancos movem-se para o lado que tem números maiores.

Um peão é geralmente uma unidade fraca, mas temos que 8 deles que podemos ser usados para construir um muro defensivo de peões. Com essa estratégia, um peão defende os outros. Um peão está seguro quando outro peão pode capturar uma unidade onde ele está. Temos apenas peões brancos no tabuleiro de xadrez, vários deles.
